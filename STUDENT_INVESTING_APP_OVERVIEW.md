# Student Investing Education Gaming App — Overview

A cross-platform gamified investing education app for students (ages 13–25).
Learn real investing using virtual money and real market data — no financial risk.

---

## Platforms

| Platform | Stack | Distribution |
|----------|-------|--------------|
| iOS | React Native + Expo | App Store |
| Android | React Native + Expo | Google Play |
| Web | Next.js | Browser |

---

## Core Game Loop

**Learn → Practice → Challenge → Compete → Earn Rewards → Level Up → Repeat**

---

## Key Features

1. **Paper Trading Simulator** — real-time stock/ETF/crypto/bond data, all virtual money
2. **Skill Tree Curriculum** — Beginner → Expert modules (stocks, ETFs, options, crypto, bonds)
3. **XP & Leveling** — 10 investor levels from Rookie to Legend
4. **Badges & Achievements** — 20+ unlockable badges for key milestones
5. **Class & School Leagues** — teachers create private leagues; semester-long seasons
6. **AI Investing Coach** — powered by Claude API for personalized Q&A and portfolio feedback
7. **Teacher Dashboard** — progress tracking, curriculum reports, class management

---

## Tech Stack

### Mobile
- React Native + Expo
- Zustand (state management)
- React Query (data fetching)
- Victory Native (charts)

### Web
- Next.js 14
- Tailwind CSS
- Recharts

### Backend
- Node.js / Express
- Python / FastAPI
- PostgreSQL
- Redis

### Data & AI
- Alpaca Markets API (stocks)
- CoinGecko (crypto)
- NewsAPI
- Anthropic Claude API (`claude-sonnet-4-6`)

### Infrastructure
- AWS ECS + RDS + ElastiCache
- Cloudflare CDN
